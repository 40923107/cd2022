uarm 一次定位到方塊並進行取放

檔案下載：40923107_xyz控制_uarm_pick and place


按 z：一次吸到方塊
按 x：上一步吸完並放下後，按 x 可再次吸取
按 y：回原高度 ( x+z 的移動高度)

新增之程式碼

            if (auxiliaryData[1]==string.byte("z")) then
                rotation2 = rotation2 - 13.8*deg
                rotation3 = rotation3 + 14.8*deg
                sim.setJointTargetPosition(axis3, rotation3)
                sim.setJointTargetPosition(axis2, rotation2)
            end -- if z
            
            if (auxiliaryData[1]==string.byte("y")) then
                rotation2 = rotation2 + 17.8*deg
                rotation3 = rotation3 - 17.8*deg
                sim.setJointTargetPosition(axis3, rotation3)
                sim.setJointTargetPosition(axis2, rotation2)
            end -- if y

            if (auxiliaryData[1]==string.byte("x")) then
                 rotation2 = rotation2 - 4*deg
                 rotation3 = rotation3 + 3*deg
                 sim.setJointTargetPosition(axis2, rotation2)
                 sim.setJointTargetPosition(axis3, rotation3)
            end -- if x
